Survival analysis

Survival analysis corresponds to a set of statistical methods used to evaluate the time for an event to occur. The event can be for example the death, the realapse or the recovery.

The survival function S(t) is for a fixed t , the probability to survive until t, i-e :

S(t) = P(X > t);

with t < 0 and X the survival duration. We can also precise the density function, precised by:

f(t) = dS(t)dt

This function can be represented by survival curves according to the time.
