Survival analysis

Survival analysis corresponds to a set of statistical methods used to evaluate the time for an event to occur. The event can be for example the death, the realapse or the recovery. Thereafter, we will continue with the example of the death as an event and the survival time, the time between the diagnosis and the death.

The survival function S(t) is for a fixed t , the probability to survive until t, i-e :

S(t) = P(X > t);

with t < 0 and X the survival duration. We can also precise the density function:

f(t) = dS(t)dt

This function can be represented by survival curves according to the time.

Moreover, survival analysis is characterised by the censorship. We call censorship the fact that we don't observe the even of interest on the individual. Right censorship is the most commun type of censorship. In this case, we know when the disease was detected but we don't know when he will died because at the last follow up at t days , the patient was alive. So, we know that I lived more than t days but we don't know exactly how long.

Survival analysis can be use to predict the efficiency of a treatment or the seriousness of a disease

An important concept is the instantanious risk. It is defined as the probability for an event in the time interval [t + Δt] knowing that the event didn't appended before t. It allows to define the risk function (or hazard function) h(t) thats appears to be a risk indicator:

$$h(t) = \lim_{\Delta t \to 0} \frac{Prob[t \leq T < t + \Delta t|T \geq t]}{\Delta t} $$

We can also define the cumulated risk H(t) defined by:

$$H(t) = \int_{0}^{t}h(s)ds $$

And, thanks to the conditional probilities theorem,

$$h(t) = \frac{f(t)}{S(t)}$$

Thanks to the equations above:

$$H(t) = log[S(t)]$$

Estimate one of these functions allow to estimate other ones. Several methods exist to estimate this function and particularly the survival function.

  • Parametrics methods : need hypothesis on the survival distribution time. Example: the Weibull model.
  • Non-parametrics methods: use when no hypothesis can be done on the survival distribution time. Example: Kaplan Meier and Nelson Aalen methods.
  • Semi-parametrics methods: Example: the Cox model.

In addition of this statistical analysis models, it exists machine learning methods, modified to be applied to survival analysis. First, we will see the statistical methods, after the machine learning methods and finally, deep learning methods (type of machine learning methods).

The paper of Ping Wang et al., explain with more details survival analysis but most of all have a great figure showing the different survival analysis methods [2].

Statistical methods for survival analysis

Parametrics methods

In a parametric survival model, as in classical regression models, one tries to relate an event to a number of explanatory variables. This model is based on the assumption that survival times follow a distribution fixed a priori. It applies to situations where a delay in the occurrence of an event is studied. One of the most widely used parametric models in the medical field is the Weibull model.

We will not go into detail about parametric models but you can find more details in [1] and [3]
