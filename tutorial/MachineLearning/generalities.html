Machine learning

Statistical or machine learning ?

We saw in the survival analysis chapter that there are statistical learning and machine learning methods. However, what are the differences between the two? The first major difference is that statistical learning formalises a relationship between variables whereas machine learning learns from data without giving clear programmed instructions. Furthermore, statistical learning requires assumptions such as distribution or collinearity. Secondly, machine learning can use a very large amount of data where statistical learning has to keep the database size small. Finally, machine learning requires less model knowledge than statistical learning [machine learning vs statistical learning].
The use of one method or another will therefore depend mainly on your data and your knowledge of it. For example, in the case of a large amount of data, machine learning methods may be the best choice, but when the amount is small, statistical methods could be more efficient in terms of computation time and results.

Supervised or unsupervised ?

In the case of supervised learning, the classes are known and the goal is to predict the new class (in the case of classification for example). Application example: Classifying pictures of animals according to their species.
Non exaustive list of methods:

  • Boosting methods
  • Support-Vector Machines (SVM)
  • Linear Regressions
  • Logistic regression
  • Naive Bayes
  • Decision trees
  • K-nearest neighbor algorithm
  • Neural networks (Multilayer perceptron)

Conversely, in the case of unsupervised learning, the classes are not known but we wish to constitute homogeneous groups thanks to common patterns. Application example: segmentation of images;
Non exaustive list of methods:

  • K-means clustering
  • Dimentionality reduction
  • Neural networks
  • PCA (Principal Componant Analysis)
  • Hierarchical clustering
  • Distribution models
  • Independent Component Analysis

Finally, there is semi-supervised learning. These are models using a set of labelled and unlabelled data. Using this unlabelled data in addition to the labelled data can greatly improve the results. Also, when the database is very large it is too tedious to label everything by hand. Thus, semi-supervised learning will save time while using all the data.
